# RTAS Web

Primary user-facing frontend for RTAS Studio AI — Next.js 14 app with BFF API routes, Prisma, NextAuth, and studio UI.

All studio/marketing/auth pages and user API routes live here. Admin operations UI is in `apps/admin`; GPU generation API is in `apps/backend`.

## Dev

```bash
npm run dev:web
```

## Shared packages

- `@rtas/shared` — types, credits, categories, legal content, pipeline constants
