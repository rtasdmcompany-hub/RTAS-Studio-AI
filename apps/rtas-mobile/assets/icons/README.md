# Icon masters — drop zone

Place your RTAS premium logo here before running `npm run mobile:assets`.

## Easy mode (one file)

| File | Size | Notes |
|------|------|-------|
| `logo-master.png` | **1024×1024 px minimum** | Centered logo on transparent or solid background |

Also accepted names: `logo.png`, `rtas-logo.png`, `icon-master.png`  
Formats: **PNG**, **JPG**, **SVG**

## Full-control mode (adaptive icon)

| File | Size |
|------|------|
| `icon-master.png` | 1024×1024 px minimum |

Pair with `../splash/splash-master.png` for separate splash artwork.

## Optional dark mode

| File | Purpose |
|------|---------|
| `logo-dark-master.png` | Dark-mode logo variant for splash/icon layers |
