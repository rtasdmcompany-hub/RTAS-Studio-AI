# Splash masters — drop zone

Required only for **full-control** mode (when using separate icon + splash artwork).

## Full-control splash

| File | Size | Notes |
|------|------|-------|
| `splash-master.png` | **2732×2732 px minimum** | Cinematic RTAS splash — logo centered, `#0a0a0f` background recommended |

Also accepted names: `splash.png`, `rtas-splash.png`  
Formats: **PNG**, **JPG**, **SVG**

## Easy mode shortcut

If you only have one logo file, drop it in `../icons/logo-master.png` — splash screens are generated automatically from that single asset.

## Optional dark splash

| File | Purpose |
|------|---------|
| `splash-dark-master.png` | Dark-mode splash variant (iOS / supported Android layers) |
