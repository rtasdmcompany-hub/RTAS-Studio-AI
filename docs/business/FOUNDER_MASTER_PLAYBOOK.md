## 2. Non-negotiable truth rules

1. Use only **Verified** pricing: Tester $5/30s/5d · Standard $89/2000s · Premium $249/2000s.  
2. Say “checkout pending / confirming” if MoR is not green — never bury it.  
3. Never invent customers, MRR, ARR, logos, partnerships, certifications, or awards.  
4. Identity Preservation = **authorized** content only. No deepfake / face-swap marketing.  
5. Label scenarios as **Planned / Illustrative / Recommended**.  
6. Prefer “not yet measured” over a guessed number.

## 4. Decision framework

| Decision type | Default | Escalate / pause when |
|---------------|---------|------------------------|
| Pricing display / live SKUs | Keep Verified tiers only | Changing amounts needs finance + legal check |
| Paid ads spend | Off until MoR conversion proven | Kill if CAC unmeasurable or checkout fails |
| Discounting | Rare; time-boxed; logged | Never perpetual unpublished “custom” without terms |
| Enterprise pilots | Design-partner first | Don’t invent SLA/SSO you can’t deliver |
| Fundraising | Optional | Don’t raise to hide MoR failure |
| M&A inbound | Listen; no exclusivity by default | Counsel before LOI / no-shop |
| AUP edge cases | Refuse misuse | Prefer lost deal over Paddle risk |

## 9. Personal risk controls

1. **Single-point failure:** Document passwords in a vault; MFA everywhere; one backup operator path if possible.  
2. **Vendor concentration:** Fal, Vercel, Supabase, Paddle — know fallbacks ([`VENDOR_MANAGEMENT.md`](../operations/VENDOR_MANAGEMENT.md)).  
3. **Burn:** Track personal/operator runway separately from product ambition.  
4. **Reputation:** One AUP violation can erase months of MoR work.  
5. **Health:** Protect sleep around Severity-1 incidents; defer vanity launches.

## 11. What “good” looks like in 90 days (Recommended, not forecast)

- Checkout works for target markets you sell into.  
- You can state real (even small) collection and conversion numbers.  
- Pipeline is logged, not remembered.  
- Zero AUP marketing drift.  
- You know whether to stay bootstrap or open a raise.

**Not required for “good”:** SOC 2, affiliate program live, enterprise logos, or a high ARR claim.
