# Promotion Guidelines

## Design Rules
- Preserve RTAS Studio AI as a premium SaaS product
- Use premium cards, rounded corners, subtle glass, soft shadows
- Keep copy concise and benefit-led
- Avoid anything that resembles display advertising

## Prohibited Patterns
- Popups
- Banner-ad stacks
- Blinking or flashing UI
- Autoplay media
- Third-party tracking pixels or scripts
- Promotions inside login, registration, checkout, password reset, error, or render/progress views

## Placement Rules
- Homepage: announcement strip only
- Dashboard: recommendation/sidebar style only
- Studio: recommendation cards only when not in progress state
- Billing/Credits: upgrade guidance
- Docs/Learning/Blog: educational and partner-ready recommendations
- Enterprise: high-intent enterprise CTA only

## Copy Rules
- Lead with value, not hype
- Label partner inventory clearly: `Partner` or `Sponsored Partner`
- Do not over-promise pricing, support, performance, or outcomes
- Keep CTA labels direct: `Compare plans`, `Open learning center`, `Contact enterprise sales`

## Analytics Rules
- Record only real interactions
- Start from zero
- Never fabricate CTR, conversion, or revenue values
- Respect existing first-party analytics posture
