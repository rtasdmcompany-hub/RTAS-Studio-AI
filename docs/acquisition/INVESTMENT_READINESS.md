## 4. What “good enough” looks like before scaling outreach

**Minimum fact pack (investment exploration):**

1. Written MoR/checkout status (green or explicitly yellow with mitigation).  
2. Demo that completes generation for a paid-tier path.  
3. Memorandum + deck content without fake charts.  
4. Legal URLs live.  
5. Secured data room folder with corporate placeholders filled or honestly marked N/A.  
6. Cap table draft (even if founder-only).  

**Still optional for early angels (but expected soon after):**

- First reconciled month of collected revenue  
- Simple cohort sheet (even n small)  
- GPU cost per second estimate from real invoices
