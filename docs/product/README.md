# Product overview — RTAS Studio AI

**Product:** RTAS Studio AI  
**Company:** RTAS Digital Marketing Company · RTAS Group of Companies  
**Category:** International AI video SaaS (Prompt-to-Video · Image-to-Video)

## Value proposition

Create cinematic AI video with identity-aware modes, transparent credit economics, and Merchant of Record billing — from first prompt to publish in one studio.

## Core journeys

1. **Sign up → verify email → Dashboard welcome** (`/profile?welcome=1`)
2. **Open Studio** → category wizard → generate
3. **Track credits, queue, library** on Dashboard
4. **Upgrade** via Paddle / Lemon Squeezy checkout
5. **Get help** via `/help`, `/how-to-use`, `/feedback`
