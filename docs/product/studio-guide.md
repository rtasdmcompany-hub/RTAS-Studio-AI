# Studio guide (product)

First-project path for creators. In-app: `/how-to-use` and Studio wizard.

## 60-second path

1. Open **Studio** from the header or Dashboard CTA.
2. Choose **Prompt to Video** or **Image to Video**.
3. Pick a **category** (story, music, product, etc.).
4. Fill the guided fields — draft **autosaves**.
5. Generate — watch progress; results land in **Your library** on the Dashboard.

## Empty states (by design)

| Surface | Message intent |
|---------|----------------|
| No mode selected | Start a project; link to guide |
| No recent / prompts | CTA back into creation |
| No carousel renders | Guide + refresh |
| Dashboard no projects | Autosave tip + Studio CTA |

## Tips that build trust

- Credits are visible before spend.
- Queue slots are honest when busy.
- Generation started modal points to Dashboard for progress.
- Help is always one click away in the header.
