## Press process

1. Confirm commercial path (billing + generation) before outbound press.
2. Use only Verified assets; label Placeholder gaps.
3. Send boilerplate + `/launch/assets` + contact `contact@rtasstudio.com`.
4. Log coverage only after it exists — never invent mentions.
